# Aspider 财经信息分析
## 项目介绍
该项目基于多类型数据源 + 信息分析Agent，对财经信息进行实时分析，并生成报告。
目的在于通过多信息面，分析行业板块上涨下跌趋势，对国家政策、经济动向进行解读。

## 执行流程
1. 从多方获取数据源数据
2. 依据数据源工厂汇总数据，进行数据清洗与向量化
3. 将数据整理为标准模板，存储ES
4. 定义多Agent协作分析流程，通过语义搜索关键词，从ES中获取信息数据进行政策解读、趋势分析

## 关键技术节点
1. 数据获取，通过api/爬虫方式从不同网站获取财政数据，包括但不限于定时政策数据获取，以及关键词定向搜索数据获取
2. 标准数据模板定义： 
   3. 标题
   4. 向量化标题
   5. 摘要
   4. 正文url
   5. 正文内容
   6. 向量化正文内容
   7. 发布时间
   8. 爬取时间
9. 标题+内容向量化，支持agent分析时的语义搜索获取数据
10. 多agent协作分析，主Agent主持讨论，分agent负责不同部分，实现政策分析，趋势解读，报告生成
